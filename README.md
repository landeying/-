# 钉钉小程序自定义组件整理
对在开发中使用的自定义组件做了简单的整理
# 目录结构
````
├─assets
│  ├─image 图片
│  └─style 样式
├─components 组件
│  ├─cascade 级联
│  ├─f2Interval 批量生成图表
│  ├─loading 加载loading
│  ├─noData 暂无数据
│  ├─scrollView 分页加载
│  ├─setFields 设置显示项
│  ├─table 表格
│  └─tree 树结构
└─pages 示例
    ├─cascade
    ├─f2Interval
    ├─index
    ├─loading
    ├─noData
    ├─scrollView
    ├─setFields
    └─table

````
